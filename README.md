# Веб-приложение для управления отелем на django

## Документация к API

### Получение списка комнат
### Запрос

`GET /api/rooms/`

### Параметры запроса

- `number` (необязательный): Фильтр по номеру комнаты.
- `capacity` (необязательный): Фильтр по вместимости комнаты.

### Ответ

Статус код: 200 OK

```json
[
  {
    "id": 1,
    "number": "101",
    "price": "2500.00",
    "capacity": 1
  },
  {
    "id": 2,
    "number": "102",
    "price": "2500.00",
    "capacity": 1
  }
]
```

### Создание новой комнаты
### Запрос

`POST /api/rooms/`

### Тело запроса
```json
{
  "number": "110",
  "price": "5000.00",
  "capacity": 2
}
```

### Ответ

Статус код: 201 Created

```json
{
  "id": 10,
  "number": "110",
  "price": "5000.00",
  "capacity": 2
}
```
